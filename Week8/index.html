Server and Technology Infrastructure of Google

The technical infrastructure of Google combines software development, networking, and systems engineering to provide reliable and fast services to billions of people [1]. The massive size and continuing growth of these systems requires many types of services, and Google consistently invents new technologies as the infrastructure adapts. Because of the scale, complexity, and innovation that structure these systems, Google is an ideal company to examine to understand the underlying technical aspects of web administration. This article will explore various components of Googleâ€™s tech stack and other noteworthy infrastructure as it applies to web administration.

Google uses many programming languages; however, a prominent employee has stated that the company tends to prefer C++, Java, and Python above others. [2] For example, Gmail uses Java and Google Search uses C++[3]. Because Google has so many projects, it is useful to have multiple programming languages available to accomplish different tasks. Most of the software used by Google in their tech stack was developed internally using these programming languages. This software stack includes the Google Web Server (GWS) which was created in-house and is used for their online services, several custom storage systems such as Colossus and Google F1, and indexing and search systems like TeraGoogle and Hummingbird. All of the software is designed to be fault tolerant so the entire infrastructure does not fail when hardware fails [4].

Several server types are used in the infrastructure, each serving a unique purpose. Web servers are used to execute search queries from users and retrieve hits using the index servers. To supply the index servers with pages, data-gathering servers equipped with a web crawler called GoogleBot are constantly searching the web for pages and using the Google algorithm to rank the pages. Document servers are used to actually store these pages. Ad servers and spelling servers are also used to help target advertisements to users and assist with search queries [5].
